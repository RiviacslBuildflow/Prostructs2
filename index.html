Install ProStructs on Your Device

Install as Web App

Add ProStructs to your home screen for an app-like experience with offline capabilities.

On Android (Samsung S24):

  1. Open this website in Chrome or Samsung Internet
  2. Tap the menu button (three dots)
  3. Select "Add to Home screen" or "Install app"
  4. Follow the prompts to complete installation

On iPhone:

  1. Open this website in Safari
  2. Tap the Share button
  3. Select "Add to Home Screen"
  4. Name it "ProStructs" and tap Add

(This button will appear if installation is available on your device)
